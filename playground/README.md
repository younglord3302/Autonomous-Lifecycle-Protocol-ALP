# ALP Playground

A browser-based interactive playground where users can create, validate, and visualize ALP projects.

## Planned Features

- Live `.alp` file editor with syntax highlighting
- Real-time validation against schemas
- Dependency graph visualization (React Flow)
- Workflow state viewer
- Export to YAML/JSON

## Technology Stack

- **Framework:** Next.js or Vite
- **Visualization:** React Flow
- **Editor:** Monaco Editor
